# Student Digital Addiction & Academic Impact Analysis (Power BI Dashboard)

## Project Overview
This project analyzes the impact of digital addiction on students' academic performance, mental health, lifestyle, and relationships using an interactive Power BI dashboard.

The dashboard provides insights into how excessive digital usage affects sleep patterns, academic performance, and student well-being.

## Tools Used
- Power BI
- Data Visualization
- Data Analysis

## Key Dashboard Metrics
- Total Students
- Average Digital Usage Hours
- Students Academically Affected
- Average Sleep Hours

## Dashboard Sections

### 1. Overview Dashboard
Provides a summary of key metrics including total students, average usage hours, academic impact, and sleep patterns.

### 2. Mental Health & Lifestyle
Analyzes the relationship between digital addiction score and mental health score, along with sleep patterns by age.

### 3. Academic Impact
Explores how digital addiction affects different academic levels.

### 4. Relationship Conflicts
Shows how digital addiction impacts relationship status and conflict levels among students.

### 5. Social Media Usage
Highlights the most used platforms and their relation to addiction levels.

## Key Insights
- High digital usage is linked with lower sleep hours.
- A significant percentage of students are academically affected.
- Addiction scores vary across academic levels.
- Digital addiction also correlates with relationship conflicts.

## Purpose
The goal of this project is to transform raw student behavioral data into meaningful insights that help understand the impact of digital addiction on academic and personal life.
